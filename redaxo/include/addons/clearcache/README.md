ClearCache AddOn für REDAXO 4
=============================

Cache löschen für alle Benutzer per Recht `clearcache[]` einstellbar.

Hinweise
--------

* Getestet mir REDAXO 4.5
* AddOn-Ordner lautet: `clearcache`
